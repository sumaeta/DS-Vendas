# DS-Vendas

# Sobre o projeto
https://sumaeta-dsvendas.netlify.app/

DS-Vendas é uma aplicação que mostra os dados das vendas em formato gráfico.

# Tecnologias utilizadas
## Back end
- Java
- Spring Boot
- JPA / Hibernate
- Maven


## Front end
- HTML / CSS / BOOTSTRAP / JS / TypeScript
- ReactJS
- yarn
